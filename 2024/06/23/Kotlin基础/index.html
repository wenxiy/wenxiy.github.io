什么是Kotlin

Kotlin是一种基于JVM的静态类型编程语言，拥有lambda，协程，属性。

Kotlin的特点

  • 尽可能少的样板代码
  • 官方支持的开发Android语言
  • 非空与不可空的数据类型使Kotlin减少了很多编译错误
  • 不需要迁移代码，Android Studio自带Java->Kotlin转换器

学习基础

  • 拥有一门面向对象程序设计语言的基础
  • 了解一些Kotlin，开发过一些Android项目
  • 最好会函数式编程

Hello Kotlin

  • 可以在IDEA中进行开发。
  • 定义函数，执行函数

基础语法

  • 变量声明

    使用两个关键字来声明变量。

    val:值不可以更改的变量。

    var:值可以更改的变量。

    var count: Int = 10
    val languageName : String = "Kotlin"

  • 类型推断

    Kotlin是一种静态类型的语言：类型将在编译时解析且从不改变。

  • Null安全

    Kotlin变量不能持有Null值，所以不能使一个引用值为Null。如果非要指定一个变量为Nul，可如此声明

    val languageName : String? = Null

  • 条件语句

    1. If-else

      if(count == 42){
          println("I have the answer")
      }
      val answerString: String = if (count == 42) {
          "I have the answer."
      } else if (count > 35) {
          "The answer is close."
      } else {
          "The answer eludes me."
      }

      println(answerString)

    2. when

      val answerString = when {
          count == 42 -> "I have the answer."
          count > 35 -> "The answer is close."
          else -> "The answer eludes me."
      }

      println(answerString)

      在这里与上面的不同就是没有声明类型，因为Kotlin拥有智能的类型转换功能。

      还可以用来判断是否包含对Null的引用。

  • 函数

    • 接受参数输入
    fun generateAnswerString(countThreshold:Int): String {
      val answerString = if (count>coutThreshold){
        "I have the answer"
      } else{
        "the answer eludes me"
      }
      return answerString
    }

    在此可以将 val变量和return都省略。
